# L4T-Calendar-App

Codebase for the development of a calendar app for [Lead4Tomorrow](https://lead4tomorrow.org/).

## Back-End Calendar Entry Formatting
```
{
    "1": { # month (Jan)
        "theme": "Anti-Bullying Month", # monthly theme
        "entries": {
            "1": "Brainstorm...", # daily entry
            "2": "Today is..."
        }
    }
}
```
